# Sumativa-2-Benjamin-Munoz-Benjamin-Pino-Tomas-Seguel
Pagina sobre venta especializada en juegos de mesa

Para la construcción del código de la página web se realizaron los siguientes prompts:

-"Realiza un código para crear una página de compras, en la cual puedas ingresar artículos a un carrito. La tienda se tratará de una tienda de juegos de mesa, tendrá diferentes categorías según el tipo de juego (en equipos, por turnos, cartas, de mesa, etc.). Debe tener colores que sean llamativos y diferenciados a cualquier otra tienda similar. El código debe seguir las siguientes indicaciones y requisitos (Se le adjuntó la rúbrica de la Ev.1 y 2...)"

Después de haber obtenido la base se le realizaron las siguientes incorporaciones/mejoras:

Actualización 1:
-Opción de filtros para ordenar los artículos mostrados (por precio, cantidad de jugadores y alfabéticamente).
-Ajustes de animaciones, color, fuente de texto y detalles para lograr una estética que sea más acorde a la página y sea acorde a CSS3.

Actualizacion 2:
-Sistema de inicio de sesión con cuentas (los ADMIN son capaces de modificar y agregar nuevos productos).
-Función de apartado emergente al hacer click en los artículos.

Actualizacion 3:
-Nuevos efectos de ondulación para el fondo de la pagina.
-Se ajustaron los colores de la pagina para seguir mas acorde el formato CSS3
-Se arreglaron pequeños desperfectos.
